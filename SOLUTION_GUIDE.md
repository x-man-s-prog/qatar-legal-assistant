## 🤖 2. تفعيل Claude API

### المشكلة
البحث بدون Claude → جودة أقل في إعادة الترتيب

### الحل

#### الخطوة 1: الحصول على مفتاح API

1. سجل في [Anthropic Console](https://console.anthropic.com/)
2. احصل على API key

#### الخطوة 2: إضافة المفتاح

```bash
# في مجلد main.py، أنشئ ملف .env
cat > .env << 'EOF'
ANTHROPIC_API_KEY=sk-ant-api03-xxxxx
MODEL_MAIN=claude-3-5-sonnet-20241022
MODEL_FAST=claude-3-haiku-20240307
EOF
```

#### الخطوة 3: تشغيل السكريبت

```bash
# تشغيل ملف التفعيل
python claude_api_setup.py

# سيُنشئ:
# - .env.example (قالب)
# - CLAUDE_API_SETUP.md (تعليمات)
```

#### الخطوة 4: إعادة تشغيل السيرفر

```bash
# Ctrl+C للإيقاف، ثم:
uvicorn main:app --reload
```

### النماذج المتاحة

| النموذج | السرعة | الجودة | التكلفة |
|---------|--------|--------|----------|
| claude-3-5-sonnet-20241022 | متوسط | ★★★★★ | مدفوع |
| claude-3-haiku-20240307 | سريع | ★★★☆☆ | مجاني* |

**مجاني في plan Claude (100 رسالة/دقيقة)_

### التحقق

curl http://localhost:8000/api/v1/health

## ❓ أسئلة شائعة

### Q: ماذا لو فشل السكريبت؟
A: تحقق من:
1. السيرفر يعمل (`curl http://localhost:8000/api/v1/health`)
2. Ollama يعمل (`curl http://localhost:11434/api/tags`)
3. قاعدة البيانات متصلة

### Q: هل يمكنني تشغيل السكريبتات معاً؟
A: نعم، لكن يفضل ترتيبها:
1. `fix_missing_embeddings.py` (مرة واحدة)
2. `integration_tests.py` (للتحقق)
3. `ollama_monitor.py` (مستمر)

### Q: كيف أعرف أن كل شيء يعمل؟
A: شغّل:
```bash
python integration_tests.py
```
يجب أن تكون جميع الاختبارات ناجحة (✓).
