## 5. تحسينات مسار البحث (5.1–5.3)

### 5.1 — تمرير `domain` للبحث
```python
# query_router.py — قبل
chunks = await search(queries, key_terms, 5)

# بعد
_search_domain = (_sf_domain or cot.get("law_domain", "") or "").lower() or None
chunks = await search(queries, key_terms, 5, domain=_search_domain)
```

**الأثر:** نتائج البحث الآن مُصفَّاة/مُعزَّزة بالمجال القانوني المكتشف من الاستعلام.

### 5.2 — تطبيع أسماء القوانين في المصادر
```python
# قبل
"title": ch["law_name"]

# بعد
"title": normalize_law_name(ch.get("law_name", "") or "")
```

**الأثر:** أسماء القوانين في المصادر الآن معيارية وموحّدة (مثلاً: "قانون رقم (11) لسنة 2004 بإصدار قانون العقوبات" → "قانون العقوبات رقم 11/2004").

### 5.3 — تعزيز رقم المادة
```python
# _enrich_with_fts — بعد الإضافة
from search_service import _apply_boosts, _extract_article_number
_article_num = _extract_article_number(query_text)
if domain or _article_num:
    fts_results = _apply_boosts(fts_results, domain, _article_num)
```

**الأثر:** عند ذكر "المادة X" في الاستعلام، تُعزَّز النتائج التي تطابق رقم المادة بمعامل ×1.50.
